In this camp, We will explain HTML & CSS development and provide a detailed overview of the technologies used to build web applications. Web development involves building the front-end, back-end, and database components of a web application.

We will begin by discussing the key details of the programming languages used in Basic Web development that is HTML & CSS. We will then dive in to Advanced web developing day-by-day

First and Foremost lets see about full stack development. Full stack web development involves building the front-end, back-end, and database components of a web application. This means that full stack developers are responsible for everything from designing the user interface to managing the server-side logic and database. Full stack web development, includes HTML, CSS, JavaScript, Node.js, and various front-end frameworks such as React and Angular.

Lets Divide Full stack into 3 parts

  • Frontend
  • Backend
  • Database

Technologies Used for Full Stack Development

Frontend Backend Database

We will be focusing on the Frontend that is HTML & CSS. This camp will be of 5 days Total. Daily sessions will be conducted by our college students. All the details will also be posted on WhatsApp group, Instagram and LinkedIn.
